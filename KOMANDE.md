# SQUAD Bot v3.0 — Kompletna lista komandi

## Ekonomija

| Komanda | Opis |
|---|---|
| `/novcanik` | Pregled coina i balansa |
| `/daily` | Dnevna nagrada (coini + XP) |
| `/posao` | Zaradi coina radom |
| `/transfer` | Pošalji coina drugom članu |
| `/shop` | Prodavnica nagrada |
| `/kupi` | Kupi predmet iz shopa |
| `/dodaj-novac` | Dodaj coina korisniku (OWNER) |
| `/oduzmi-novac` | Oduzmi coina korisniku (OWNER) |
| `/top-bogati` | Leaderboard najbogatijih |
